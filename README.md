# 陈沛哲 | 嵌入式软件开发工程师

个人品牌网站 — Apple/Linear 极简白灰风格，纯手工 HTML/CSS/JS 构建。

## 技术栈

- HTML5 + CSS3 + Vanilla JS（ES6+）
- 无框架、无构建工具
- Inter + JetBrains Mono 字体（Google Fonts）
- Devicon CDN 工具图标

## 页面

| 页面 | 说明 |
|------|------|
| `index.html` | 首页 — 个人信息 + 技能图标 + 获奖荣誉 + 四个分类卡片 |
| `linux-driver.html` | Linux 驱动与应用开发 — 5 个项目（RK3588/Sanbot/点胶/瞄准/图传） |
| `vision-ai.html` | 视觉图像处理与边缘AI — 3 个项目（搬运/激光追踪/纽扣检测） |
| `qt-tool.html` | Qt 上位机 — 2 个项目（示波器/BLE 蓝牙） |
| `wireless.html` | 无线通信 — 2 个项目（Find My/平衡小车） |
